# SwapSkill Frontend

The frontend web application for SwapSkill, providing an intuitive interface for users to exchange skills and connect with others.

## Key Features
- **Responsive UI**: Beautiful and responsive design using Tailwind CSS.
- **Smooth Animations**: Interactive UI components powered by Framer Motion.
- **Data Fetching**: Fast and reactive data handling with SWR and Axios.
- **Dark Mode Support**: Seamless theme switching with next-themes.
- **Toast Notifications**: Real-time user feedback with react-hot-toast.

## Tech Stack
- **Framework**: Next.js 16, React 19
- **Styling**: Tailwind CSS, clsx, tailwind-merge
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Data Fetching**: SWR, Axios
